# visual-editor Plugin

MVP fuer einen Block-basierten Editor-Workflow.

Enthalten:

- Admin-Seite unter `Visual Editor` (Menue)
- `POST /visual-editor/blocks/parse` (Markdown -> Blocks)
- `POST /visual-editor/blocks/render` (Blocks -> Markdown)
- Health-Route: `GET /visual-editor/health`

Status: Beta / Bruecke fuer spaetere Editor-Integration in die Content-Ansicht.
